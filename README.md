# 🏦 Bank Management System

This is a CLI-based Bank Management System built using Python.

## 🚀 Features
- Create account with unique account number  
- Deposit & Withdraw money  
- Transfer money between accounts  
- View account details  
- Track transaction history  

## 🛠️ Tech Used
- Python  
- Data Structures (Dictionary, List)  

## 📊 Description
This project simulates a basic banking system where users can manage accounts and perform transactions.

## ▶️ How to Run
1. Run the script:
   python filename.py
